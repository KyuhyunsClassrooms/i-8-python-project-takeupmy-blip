## 3. 입력-처리-출력 구조

### 입력

- 이용자가 플레이 할 난이도 선택
- 이용자가 입력하는 퀴즈의 정답
- 게임을 다시 시작할지 여부

### 처리

- 이용자가 선택한 난이도에 따른 문제들을 리스트에서 찾기
- 조건문을 활용해 정답 여부 판단 후 점수 입력
- 반복문을 사용하여 난이도에 따른 퀴즈 하나씩 표기

### 출력

- 매 문제마다 정답여부 출력
- 퀴즈 최종 화면에서의 총 점수

---

## 4. 2차원 리스트 설계

내 프로그램에서 사용할 2차원 리스트의 이름:qize_show


```python
qize_show=[
    ["과학", "최초의 컴퓨터의 이름은?","애니악"]
    ["과학", "노벨이 발명한 폭탄의 이름은?","다이너 마이트"]
    ["과학","빛은 굴절한다","O"]
]

```

### 한 행의 의미
하나의 퀴즈 문제에 대한 정보

### 각 열의 의미

| 열 번호 | 의미 | 예시 |
|---:|---|---|
| 0번 열 | 퀴즈 난이도 | 쉬움, 보통 |
| 1번 열 | 이용자에게 보여줄 퀴즈 내용 | 최초의 컴퓨터의 이름은 무엇인가. |
| 2번 열 | 입력과 비교할 실제 정답 | 애니악 |
| 3번 열 |  |  |

## 5. 함수 설계

| 함수 이름 | 역할 | 입력값 | 반환값 또는 출력 |
|---|---|---|---|
| menu | 난이도 선택 창 | 없음 | 선택한 난이도 이름 |
| quiz | 난이도에 따른 문제 출제 후 채점 | 난이도 이름 | 정답 개수, 총 문제 수 |
| result| 맞친 총 개수 알림 | score,total | 화면 출력 |
